## Clone Project into your machine
# Run 
    composer update
# Run 
    npm install -D tailwindcss postcss autoprefixer
    npx tailwindcss init -p
# Run 
    php artisan migrate
    php artisan db:seed
    php artisan key:generate
    npm run dev
    php artisan serve
You can explore project permissions and roles are added  by PermissionSeeder
